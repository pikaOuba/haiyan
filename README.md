# react
react spa to show repo name etc

## 项目下载到本地后，使用npm  install 下载相关依赖
## 或者用这个安装依赖
npm install -g cnpm --registry=https://registry.npm.taobao.org 
## npm run dev  运行项目  
## npm run build  项目打包
